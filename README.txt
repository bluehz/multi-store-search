Multi-Store Search PWA

Files included:
- index.html        (Main app page)
- manifest.json     (PWA config)
- sw.js             (Service worker for offline caching)
- icon-192.png      (App icon 192x192 px)
- icon-512.png      (App icon 512x512 px)

Instructions:
- Replace icon PNGs with your own branding if desired.
- Edit manifest.json to update app name, description, or start_url if you rename index.html.
- Upload all files to your GitHub repository root.
- Enable GitHub Pages on the repository (branch main, root folder).
- Visit your site URL (https://yourusername.github.io/your-repo-name/)
- Add to Home Screen on your mobile device for PWA experience.

Enjoy!
